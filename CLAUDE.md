# やりたいことアプリ

SNSで出会った美術展やご飯屋さんを3秒でスクラップし、ちょうどいいタイミングでリマインドが届くiOSアプリ。「人生でやり残したことを減らす」が目的。

## ユーザーとの話し方（厳守）

kanamiさんはUIUXデザイナーで、アプリ開発（コーディング・Xcode・Git等）は未経験。

- 専門用語には必ず一言のたとえを添える（例：ビルド＝コードを組み立てて動くアプリの形にすること）
- 手順は1ステップずつ。一度に複数の操作を頼まない
- デザイン（色・余白・動き・言葉選び）はkanamiさんの専門領域。勝手に決めず、実装上の制約だけ伝えて判断は本人に委ねる
- やりとりは日本語

## 決定事項（2026-07-07）

1. **開発の順番**：HTMLモック（prototype/index.html）でUXを固めてから SwiftUI に移植
2. **入力方法**：共有シート対応をMVPに含める（TwitterやインスタのシェアボタンからこのアプリにURLを送れる）
3. **日付の読み取り**：AI（Claude API）でページから開催日を抽出。読み取り結果は候補として表示し、手動で直せるUIにする
4. **リマインド設計**（kanamiさんのUX判断）：
   - 開始日の少し前（例：3日前）
   - 終了が近づいたとき（例：1週間前）
   - 日付がないもの（ご飯屋さん等）は定期的に思い出させる（例：1ヶ月後に「これ行った？」）
5. MVPを小さくリリースしてから改善を重ねる
