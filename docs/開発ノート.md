## フェーズ2の技術方針メモ（着手時にkanamiさんに説明しながら進める）

- SwiftUI、外部ライブラリなし。データ保存はSwiftData（iPhoneの中のノートに書き留めるような、端末内保存の仕組み）
- 共有シート対応は「Share Extension」（アプリ本体に付ける小さな受付窓口）＋「App Groups」（本体と窓口が同じ引き出しを使うための設定）が必要
- リマインドはローカル通知（ネット不要でiPhone自身が鳴らす通知）
- 画面構成：ホーム一覧 / 保存・編集シート / 詳細 の3つから始める
- `ScrapRepository`（データの窓口役）を1枚挟み、保存先やAIの呼び先を将来差し替えられるようにする
